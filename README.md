# Unity-3D-RPG
멀티플레이 기반 3D RPG
<br><br>

## 프로젝트 소개
Unity 엔진을 이용하여 제작한 멀티플레이 기반 3D 액션 RPG 입니다.
<br><br>

- 설계 : 2023.04.10 ~ 2023.05.12
- 구현 : 2023.05.13 ~ 2023.11.27

- 개발 툴: Unity 2022.1.23f1
- 서버 환경: AWS EC2 (Ubuntu)
- 데이터베이스: MySQL
- 멀티플레이어 SDK: Photon PUN2
